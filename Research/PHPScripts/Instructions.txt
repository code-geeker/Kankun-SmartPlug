dasygenis_kankun_control.sh and relaycontrol.sh and smartsocket.php.txt and watchdog.sh. Scripts to remotely control via web interface the Smart socket Kankun. Installation: SSH into the embedded system of smart socket as root, and issue these commands:
cd /root ; 
wget http://arch.icte.uowm.gr/mdasyg/misc/dasygenis_kankun_control.sh ; 
chmod 755 dasygenis_kankun_control.sh ; 
wget http://arch.icte.uowm.gr/mdasyg/misc/relaycontrol.sh ; 
chmod 755 relaycontrol.sh
wget http://arch.icte.uowm.gr/mdasyg/misc/watchdog.sh ;
chmod 755 watchdog.sh
Insert the line:
/root/dasygenis_kankun_control.sh &
at /etc/rc.local before exit 0. Copy the smartsocket.php.txt to your PHP web server ommiting the txt extension. Edit dasygenis_kankun_control.sh and modify the [serverIP], [name] and [psk] fields. Edit smartsocket.php and modify the [psk] {to be the same as previous}.
